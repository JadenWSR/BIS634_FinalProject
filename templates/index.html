YALE BIS634 Final Project

The project is mainly about analyzing the COVID-19 vaccine tweets dataset.

This local Web application is built with Python flask, along with some HTML and CSS.

There are 7 app routes

  • @app.route('/'): Home Page
  • @app.route('/data'): Data Page.
  • @app.route('/wordcloud'): Word Cloud Analysis Page.
  • @app.route('/timeseries'): Timeseries Analysis Page.
  • @app.route('/model_page'): Model Page without Output Result.
  • @app.route('/model_result'): Model Page with Output Result.
  • @app.route('/contact'): Contact Page

and 7 APIs.

  • @app.route('/wordcloud_api/<string:vax_name>'): Returns JSON-encoded data generating a word cloud given a vaccine brand.
  • @app.route('/wordcloud_api/senti/<string:senti>'): Returns JSON-encoded data generating a word cloud given a sentiment.
  • @app.route('/wordcloud_api/country/<string:country>'): Returns JSON-encoded data generating a word cloud given a country name.
  • @app.route('/timeseries_api/varianceVStime/<string:vax_name>'): Returns JSON-encoded data generating a timeseries graph given a vaccine brand.
  • @app.route('/timeseries_api/varianceVStime_Country/<string:country>'): Returns JSON-encoded data generating a timeseries graph given a country name.
  • @app.route('/timeseries_api/SDT/<string:senti>'): Returns JSON-encoded data generating a bar graph given a sentiment.
  • @app.route('/model_api/<string:user_text>'): Returns JSON-encoded data generating the predicted results from the model given a user input.
